# 월간 회의자료 자동화 스크립트

이 프로젝트는 매주 작성된 주간 회의자료(PPTX)를 취합하여 하나의 월간 보고서(PPTX 및 DOCX)를 자동으로 생성합니다.

## 주요 기능
- **가로형 주간 PPT -> 세로형 월간 PPT**: 입력받은 가로형 PPT 텍스트를 추출하여 A4 세로 규격의 월간 보고서로 변환합니다.
- **DOCX 보고서 생성**: 동일한 내용을 워드 문서 형태로도 제공합니다.
- **가변적 주차 처리**: 4주차 또는 5주차 등 폴더 내의 파일을 자동으로 인식합니다.
- **사용자 선택형 출력**: 실행 시 PPTX, DOCX 중 원하는 형식을 선택할 수 있습니다.

## 설치 방법
Python 3.10 이상이 설치되어 있어야 합니다.

```bash
pip install -r requirements.txt
```

## 사용 방법

### 방법 1: 데스크톱 위젯 (추천)
드래그 앤 드롭으로 간편하게 사용할 수 있는 GUI 모드입니다.

```bash
python monthly_widget.py
```
1. 위젯이 실행되면 주간 PPTX 파일들을 한꺼번에 드래그하여 위젯 중앙에 놓습니다.
2. 파일 목록을 확인하고 '변환 시작'을 누릅니다.
3. 원하는 형식을 선택하면 결과물이 생성되고 자동으로 폴더가 열립니다.

### 방법 2: 터미널 스크립트
명령어 기반으로 실행하는 모드입니다.

```bash
python generate_monthly.py --month 2025-05
```

3. 터미널에 나타나는 안내에 따라 출력할 파일 형식을 선택합니다 (1: PPTX, 2: DOCX, 3: Both).
4. `output/` 폴더에서 생성된 결과를 확인합니다.

## 설정 변경
`generate_monthly.py` 상단의 `CONFIG` 섹션에서 다음 항목을 수정할 수 있습니다:
- `input_dir`: 입력 파일 경로
- `output_dir`: 결과 파일 저장 경로
- `pptx_size`: PPT 슬라이드 크기 (기본 A4 세로)
- `file_prefix`: 주간 보고서 파일명 시작 단어
